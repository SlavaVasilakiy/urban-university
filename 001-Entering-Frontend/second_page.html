HTML (HyperText Markup Language)
  • Язык разметки:
    HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. Он используется для структурирования контента на веб-странице, включая текст, изображения, ссылки и другие элементы
  • Теги и элементы:
    HTML состоит из тегов, которые заключены в угловые скобки (< >). Теги обычно идут парами: открывающий тег (например, <p>) и закрывающий тег (например, </p>). Некоторые теги, такие как <img>, являются самозакрывающимися.
  • Атрибуты:
    Теги HTML могут иметь атрибуты, которые предоставляют дополнительную информацию о элементе. Атрибуты указываются внутри открывающего тега и состоят из имени и значения (например, <a href="https://example.com">).
  • Документ HTML:
    Структура HTML-документа обычно начинается с декларации <!DOCTYPE html>, за которой следует элемент <html>, содержащий два основных раздела: <head> (метаданные документа) и <body> (содержимое документа).
CSS (Cascading Style Sheets)
  • Стилизация веб-страниц:
    CSS (Cascading Style Sheets) используется для описания внешнего вида и форматирования HTML-документов. С помощью CSS можно задавать стили для элементов, такие как цвет текста, шрифты, отступы, границы, фоны и многое другое.
  • Каскадность и наследование:
    CSS поддерживает каскадность, что означает, что стили могут наследоваться от родительских элементов к дочерним. Это позволяет создавать более гибкие и управляемые стили. Приоритет стилей определяется их специфичностью и порядком объявления.
  • Медиа-запросы и адаптивный дизайн:
    CSS включает в себя медиа-запросы, которые позволяют изменять стили в зависимости от характеристик устройства, таких как ширина экрана, ориентация и разрешение. Это важно для создания адаптивного дизайна, который хорошо выглядит на различных устройствах, от мобильных телефонов до настольных компьютеров.
  • Модули и препроцессоры:
    Современные подходы к CSS включают использование модулей и препроцессоров, таких как SASS или LESS. Эти инструменты добавляют функциональность, такую как переменные, вложенные правила и миксины, что делает код более организованным и повторно используемым.
JavaScript
  • Динамическая типизация:
    JavaScript является языком с динамической типизацией, что означает, что тип переменной определяется во время выполнения, а не во время компиляции. Это позволяет разработчикам писать более гибкий код, но также требует внимательности, чтобы избежать ошибок, связанных с типами данных.
  • Асинхронность и обработка событий:
    JavaScript поддерживает асинхронное программирование, что позволяет выполнять задачи, не блокируя основной поток выполнения. Это особенно полезно для работы с сетевыми запросами, таймерами и пользовательскими событиями. Основные механизмы асинхронности включают колбэки, промисы и async/await.
  • Работа в браузере и на сервере:
    Изначально JavaScript был разработан для работы в браузере, где он используется для создания интерактивных веб-страниц. Однако с появлением Node.js, JavaScript также стал популярным для серверного программирования, что позволяет использовать один язык для разработки как клиентской, так и серверной части приложения.
  • Объектно-ориентированное программирование (ООП):
    JavaScript поддерживает объектно-ориентированное программирование, что позволяет создавать объекты и классы, использовать наследование и инкапсуляцию. В ES6 (ECMAScript 2015) были введены классы, которые упростили синтаксис ООП и сделали его более понятным и удобным для разработчиков.
